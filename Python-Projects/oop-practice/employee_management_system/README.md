# 🧑‍💼 Employee Management System (Python + Tkinter)

A simple desktop Employee Management System built using Python and Tkinter GUI.

## 🚀 Features

- Add Employee
- Prevent Duplicate ID
- Search by ID
- Update Employee
- Delete Employee
- Display All Employees
- Error Handling & Validation

## 🛠 Technologies Used

- Python
- OOP (Object-Oriented Programming)
- Tkinter (GUI)
- Exception Handling

## 📌 Project Structure

- Employee Class
- Manager Class
- Developer Class
- Company Class
- Tkinter GUI Interface

## 🎯 What I Learned

- Applying OOP concepts in a real project
- Building GUI applications
- Data validation & error handling
- Managing data inside classes
- Structuring a clean project
